# PACT Public ADR Index

This directory records public, implementation-neutral architecture decisions for the PACT specification surface.

Status legend:

1. `Accepted`: normative decision for the public specification baseline.
2. `Draft`: proposed decision, not yet part of normative baseline.
3. `Superseded`: replaced by a later ADR.

Current ADR set:

1. `ADR-000`: Public scope and non-goals.
2. `ADR-001`: Governance safety invariants.
3. `ADR-002`: Artifact contract and conformance fixture model.
4. `ADR-003`: Extension model and reference vertical packs.

These ADRs define specification constraints only. They do not prescribe runtime implementation details.
