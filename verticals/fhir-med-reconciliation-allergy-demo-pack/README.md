# FHIR Medication Reconciliation + Allergy Demo Pack

Non-normative public companion demo pack grounded to HL7 FHIR R5 resources used in medication reconciliation and allergy safety workflows.

## Source Grounding

Primary sources:

1. https://www.hl7.org/fhir/medications-module.html
2. https://www.hl7.org/fhir/list.html
3. https://www.hl7.org/fhir/allergyintolerance.html
4. https://www.hl7.org/fhir/medicationstatement.html

## Included Resource Surface

1. List (curated snapshots/changes for medication reconciliation)
2. AllergyIntolerance (risk statements relevant to medication safety)
3. MedicationStatement (patient-reported/compiled medication usage)
4. MedicationRequest (order context for reconciliation)

## Status

1. Public candidate demo pack for interoperability and governance review.
2. Non-normative and does not replace HL7 implementation guides.
