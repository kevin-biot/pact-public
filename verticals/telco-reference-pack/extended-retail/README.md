# Telco Extended Retail Companion

This companion set adds a richer telco retail ontology surface for public review.

## Purpose

1. Show a broader intent surface (SIM swap, port-in, top-up, plan upgrade, balance check, KYC verify).
2. Provide richer lexical artifacts (`patterns`, `thesaurus`, `thesaurus-local`).
3. Demonstrate jurisdiction overlays (`overlay-eu`, `overlay-om`).
4. Keep baseline `telco-reference-pack` signed artifacts unchanged for existing fixture compatibility.

## Status

1. Non-normative companion artifacts.
2. Not part of the signed OCI-1 baseline for `verticals/telco-reference-pack`.
3. Intended as a candidate preview surface for future public pack promotion.
