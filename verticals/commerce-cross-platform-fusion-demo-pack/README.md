# Cross-Platform Commerce Fusion Demo Pack

Bounded, deterministic cross-platform route demo linking:

1. Open Banking intent ingress and status/cancel semantics
2. Shopify checkout function control points
3. ISO 20022 rail submission, status, and return semantics

## Primary narrative

1. Ingest open banking payment initiation intent.
2. Normalize to ISO 20022 initiation semantics.
3. Apply Shopify checkout transform/discount/payment/validation controls.
4. Submit credit transfer and reconcile status via ISO 20022.
5. Sync status back to open banking client.
6. Support post-submission compensation via cancel/return path.
