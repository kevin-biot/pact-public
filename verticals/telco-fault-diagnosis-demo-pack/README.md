# Telco Fault Diagnosis Demo Pack

Non-normative public demo pack for fault diagnosis governance.

## Scope

1. Fault diagnosis task admission and validation.
2. Deterministic field policy checks for identifier/time/priority.
3. Out-of-scope escalation behavior.

## Status

1. Public candidate demo pack for interoperability and governance review.
2. Does not define TM Forum standards and does not replace IG1453.
