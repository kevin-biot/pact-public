# Open Banking AIS Read Demo

Closed-world ontology demo for account-information reads.

## Scope

1. Accounts read
2. Balances read
3. Transactions read

## Source profile intent

1. Berlin Group NextGenPSD2 AIS shape
2. UK Open Banking account and transaction shape
3. FDX account and transaction data shape

## Runtime boundary

1. Out-of-scope requests escalate.
2. Deterministic intent routing only for AIS read intents.
