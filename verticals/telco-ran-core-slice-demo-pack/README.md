# Telco RAN Core Slice Demo Pack

Non-normative public demo pack for RAN/Core/Slice intent governance.

## Scope

1. Slice attach-style intent admission and validation.
2. Deterministic checks for slice identity and managed target.
3. Out-of-scope escalation for non-domain intents.

## Status

1. Public candidate demo pack for interoperability and governance review.
2. Does not define 3GPP standards and is not a conformance profile.
