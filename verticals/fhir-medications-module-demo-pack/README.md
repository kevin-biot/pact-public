# FHIR Medications Module Demo Pack

Non-normative public demo pack grounded to HL7 FHIR R5 Medications Module.

## Source Grounding

Primary source:

1. https://www.hl7.org/fhir/medications-module.html (FHIR R5, accessed 2026-03-31)

Module scope includes medication ordering/dispensing/administration/statement,
medication knowledge resources, and immunization resources.

## Included Resource Surface

1. MedicationRequest
2. MedicationDispense
3. MedicationAdministration
4. MedicationStatement
5. Medication
6. MedicationKnowledge
7. FormularyItem
8. Immunization
9. ImmunizationEvaluation
10. ImmunizationRecommendation

## Status

1. Public candidate demo pack for interoperability and governance review.
2. Non-normative and does not replace HL7 implementation guides.
