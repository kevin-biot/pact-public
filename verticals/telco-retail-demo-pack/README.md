# Telco Retail Demo Pack

Non-normative public demo pack with a broader telco retail intent surface.

## Scope

1. SIM swap
2. Port in
3. Top up
4. Plan upgrade
5. Balance check
6. KYC verify

## Included Surfaces

1. Signed OCI-1 baseline artifacts (`pack.json`, `bundle.json`, `obt.jws`).
2. Intent convergence, patterns, and thesaurus artifacts.
3. Jurisdiction overlays for EU and Oman.
4. Candidate decision-space and extraction-summary companion artifacts.

## Status

1. Public candidate demo pack for interoperability and governance review.
2. Non-normative and does not define TM Forum or 3GPP standards.
